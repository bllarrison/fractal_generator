# fractal_generator  

Run the program and view a fractal getting generated.  

Controls:  
Mouse click and drag  - change viewing location  
C                     - Increase detail (more lines in viewspace)  
X                     - Decrease detail (less lines in viewspace  
B                     - Zoom in  
V                     - Zoom out
